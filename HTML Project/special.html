COURSE SPECIALIZATIONS

Software Engineering






About the Specialization:
The specialization in Software Engineering prepares students to become first-rate IT professionals, proficient in designing, developing and managing computing solutions. This degree program equips students for successful and rewarding careers in spefications design, coding and tesing of software.

Tasks:
1) Design and develop software systems, using scientific analysis and mathematical models to predict and measure outcome and consequences of design.
2) Analyze information to determine, recommend and plan installation of a new system or modification of an existing system.
3) Coordinate installation of software system.
4) Store, retrieve, and manipulate data for analysis of system capabilities and requirements.
5) Supervise and assign work to programmers, designers, technologists and technicians and other engineering and scientific personnel.

Career Opportunities:

  • Applications Programmer/Developer
  • Systems Analyst/Developer
  • Information Technology Instructor
  • Database Programmer/Designer
  • Information Security Engineer
  • Quality Assurance Enginner/Specialist
  • Researcher
  • System Administrator
  • Technical Write


Multimedia Arts






About the Specialization:
Students will have the opportunity to study global multimedia usability and accesibility standards, management principles for creative production, the principles for creative production, the design process, elements of visual design, information system design, instructional design, and digital imaging by specializing in the filed of Multimedia Arts and Animation. They will experice web design and content management,web animation, web video and streaming video, 3D modeling, and animation.

Tasks:
1) Multimedia artists use multiple technologies and artistic methods to create special effects, characters or scenes in film, advertisements, website design, animation or video games.
2) Multimedia artists use different technologies and elements like typography, images and music to create unique work.
3) Multimedia artists create realistic images by controlling light, color, shadow, texture and movements. While any initial ideas may be sketched by hand, they will be brought to life using computer software programs.
4) Multimedia artists create two and three-dimensional designs using modeling programs.
5) Storyboards and plot lines may also be created by multimedia artists to further develop a story in a video game or animated film.

Career Opportunities:

  • 2D/3D Designer
  • 2D Animator/Illustrator
  • Computer Graphics Designer
  • Database Administrator
  • Entrepeneur in IT Industry
  • Multimedia Designer/Developer
  • Photographer
  • Print/Desktop Publisher
  • QA Specialist
  • System Analyst/Developer
  • Technical Support Specialist
  • Web Designer/Developer


Mobile and Web Applications






About the Specialization:
Students will gain a broad foundation on user interaction and global communication using digital tecnologies by specializing in Mobile and Web Applications. They will learn usability and design principles, Software development and management, application development for cell phones, tablets, websites, handheld gadgets, and other devices. Further, students will be exposed to web design concepts and technologies, graphics and computer programming.

Tasks:
1) Using their knowledge of software engineering, mobile applications developers collaborate with analysts, other developers and project managers to determine the requirements for mobile applications.
2) Mobile applications developers primarily design and code programs that run on phones and tablets, while some may create applications for these devices and also make Web applications.
3) These developers determine the mobile program's appearance, the functions the application should perform and the integration needed with Web services.
4) Self-employed mobile developers may determine their own requirements and limitations, and consultants may work with businesses to figure out the details of a program.
5) The developer designs and codes the app. During these processes, modifications may be made until the application functions as planned.

Career Opportunities:

  • Computer Graphics Designer
  • Database Administrator
  • Mobile Applications Developer
  • Network Administrator
  • Network Engineer/Network Specialist
  • QA Specialist
  • System Analyst/Developer
  • Technical Support Specialist
  • Web Designer/Developer
